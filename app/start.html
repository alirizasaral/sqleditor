Core Insurance Sandboxes SQL-Editor

Some useful hints

With power comes responsibility. You have read and write access to the database of your box through this editor. Be careful, what you execute!

Press control-space for auto-completion of sql commands.

Execute one query at a time. Bulk-Queries are not allowed.

Consider the performance and result set size of your query. You may get gateway timeout, although your query is still being executed in the background.

Use /* */ to comment multiple lines and -- to comment the rest of a single line. Do not use #.

Some sample queries
  • SELECT * FROM ABS.TVERSION;
  • SELECT DISTINCT(anlagejahr || ' ' || nummer) AS Claims FROM ABS.TSCHADEN;
  • SELECT DISTINCT(firma || pollfnr) AS Contracts, stadium AS Phase FROM ABS.TVERTRAG WHERE pollfnr != '';
  • UPDATE ABS.TGSCHFALL SET GSTUEBERNZEIT = '0005-01-01 00:00:00.0' WHERE gf# = (SELECT gf# FROM ABS.TVERTRAG WHERE pollfnr = '8000731920');
  • UPDATE ABS.TGSCHFALL SET SPERRUSER = '';
